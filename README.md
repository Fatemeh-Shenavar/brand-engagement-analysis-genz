# Brand Engagement Analysis (Gen Z)

## Overview
This project analyzes social media engagement for Dior, Gucci, and Versace on TikTok and Instagram, focusing on Gen Z behavior.

## Key Insights
- High reach does not equal high engagement  
- Gen Z prefers interactive content over passive viewing  
- Engagement depends on content relevance, not posting frequency  

## Strategy
- Shift from broadcast to participatory content  
- Use creators and user-generated content (UGC)  
- Increase interactivity (polls, Q&A, short-form video)  

## Skills
Data Analysis | Social Media Analytics | Business Strategy
